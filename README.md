# GitHub Necessary Commands

### Push for the first time

<p>First </p>

```
git init
```

<p>2nd </p>

```
git add .
```

<p>3rd</p>

```
git commit -m "first commit"
```

<p>4th</p>

```
git branch -M main
```

<p>5th</p>

```
git remote add origin [link from HTTPS]
```

<p>6th</p>

```
git push -u origin main
```
